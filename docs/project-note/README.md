## Особенности документа

### Placeholders для визуального материала

В документе используются placeholders для рисунков, схем и скриншотов в следующем формате:

```
[PLACEHOLDER: Рисунок X.Y - Название]
Описание: Краткое описание
Содержание: Детальное описание элементов
```

Необходимо создать соответствующие визуальные материалы и вставить их в указанные места.
