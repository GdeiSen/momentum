# Momentum Platform V2

## Purpose

This document set describes the target state of Momentum as a social wellness platform for teams, active lifestyle communities, coaches, and nutrition experts.

The current repository already contains:

- authentication and user profiles
- teams and memberships
- team challenges and habits
- one chat per team
- team posts
- user dashboard and settings
- AI assistant

The target platform keeps those parts as a foundation and expands them into a broader product with:

- workspaces and multiple channels
- customizable roles and permission-based access control
- social feed and public or private activity sharing
- workout tracking
- nutrition tracking
- biometric and health data
- coach and nutritionist workflows
- integrations with external devices and health ecosystems

## Documentation Rules

- `must` means a hard implementation requirement
- `should` means a strong recommendation
- `may` means an optional extension

## Guiding Assumption

Momentum should not become a narrow habit tracker with extra screens. It should become a modular active lifestyle platform where habits, workouts, nutrition, social interaction, and expert support coexist under one coherent product model.
